# File-Handling-CRUD-Project


A Python application designed to perform Create, Read, Update, and Delete (CRUD) operations on files. This project demonstrates data persistence and exception handling without relying on an external database.

## 🚀 Features
- **Create**: Generate new text files and write data records.
- **Read**: Fetch and display specific records or full logs.
- **Update**: Search and modify existing data safely.
- **Delete**: Remove unwanted records from the storage file.

## 🛠️ Tech Stack
- **Language**: Python 3
- **Core Concepts**: File I/O (`open()`, modes like `r`, `w`, `a`), Error Handling (`try-except`)

## ⚙️ How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com
   ```
2. Navigate to the project directory and run the main file:
   ```bash
   python FileHandlingProject.py
   ```
